# 项目简介

## 核心功能
- 宠物信息展示
- 公告信息发布
- 智能客服交互
- 个人信息管理
- 密码修改
- 宠物领养流程
- 宠物日常管理
- 我的收藏功能
- 系统首页管理
- 用户管理
- 宠物分类管理
- 宠物信息管理
- 宠物领养管理
- 系统管理

## 适用场景
- 宠物领养服务机构
- 宠物爱好者社区
- 宠物相关企业
- 个人宠物领养需求

## 技术特点
- B/S架构，易于部署和维护
- MVC模式，提高代码可读性和可维护性
- SpringBoot和Vue技术栈，保证系统的高效运行和良好的用户体验
